Comando para conectar ao banco com psql:
	
	psql -h [HOST] -p [PORT] -d [DATABASE_NAME] -U [USER]

Execute o script decriação do banco:

	mapaosc.sql

Execute o script de configuração do banco:

	config.sql

Execute o script de criação das view:

	views.sql

Execute o script de criação dos índices:

	index.sql

Execute o script de criação das functions:

	functions.sql
